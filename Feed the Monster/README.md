# Feed the Monster

A friendly Phaser 3 game for young kids. Food falls from above, and players drag or tap the right snacks into the monster's mouth while avoiding rocks and foods the current monster does not want.

## Features

- Baby and Explorer modes
- Tap-to-toss and drag-to-feed controls
- New monsters with different favorite foods
- Funny chomp, bounce, shake, and celebration animations
- Hats and accessories unlocked as the player progresses

## Scripts

```bash
npm install
npm run dev
npm run build
```
